KSP KEYBOARD CONTROLS (default bindings):

FLIGHT CONTROLS:
- W/S: Pitch down/up
- A/D: Yaw left/right
- Q/E: Roll left/right
- Shift: Increase throttle
- Ctrl: Decrease throttle
- Z: Full throttle
- X: Cut throttle
- Space: Stage (activate next stage)
- T: Toggle SAS (stability assist)
- R: Toggle RCS (reaction control system)
- F: Hold for precision control mode (fine adjustments)
- CapsLock: Toggle precision control mode

SAS MODES (click navball icons when SAS is on):
- Stability Assist: holds current orientation
- Prograde/Retrograde: locks to orbital velocity direction
- Normal/Anti-normal: locks perpendicular to orbital plane
- Radial In/Out: locks toward/away from body
- Target/Anti-target: locks toward/away from target (requires target set)
- Maneuver: locks to next maneuver node (requires node placed)
SAS modes require appropriate pilot level or probe core capability.

TIME WARP:
- Period (.): Increase physical time warp (1x, 2x, 3x, 4x) — allows physics, limited speed
- Comma (,): Decrease physical time warp
- Period (.): Also increases rails time warp when not under thrust (5x to 100000x) — no physics
- Alt + Period/Comma: Switch between physical and rails warp
- / (slash): Warp to next maneuver node
- Cannot warp at high speed in atmosphere or near terrain
- Physical warp can cause structural failures on fragile craft

MAP VIEW:
- M: Toggle map view
- Tab: Cycle focus between vessels/bodies
- Backspace: Focus on current vessel
- Double-click: Focus on a body or vessel

CAMERA:
- V: Cycle camera modes (Auto, Free, Orbital, Chase)
- Scroll wheel: Zoom in/out
- Middle mouse: Pan camera

NAVIGATION:
- Set target: click vessel/body in map view, click "Set as Target"
- Place maneuver node: click on orbit path in map view
- Delete maneuver: right-click node, click delete

OTHER ESSENTIAL CONTROLS:
- G: Toggle landing gear
- U: Toggle lights
- B: Toggle wheel brakes
- Backspace (hold): Abort action group
- 1-9: Custom action groups
- 0: Abort action group
- F5: Quicksave
- F9 (hold): Quickload
- Escape: Pause menu
- Alt+F12: Debug/cheat menu
- F1: Screenshot
- F2: Toggle UI visibility

EVA CONTROLS:
- Space: Jump (on surface) / Let go of ladder
- W/S/A/D: Move
- Shift: Climb up ladder / Jetpack up
- Ctrl: Climb down ladder / Jetpack down
- R: Toggle jetpack
- F: Interact (board vessel, grab ladder, plant flag)
- L: Toggle helmet light

EDITOR CONTROLS (VAB/SPH):
- Place parts by clicking and dragging from parts list
- Offset tool (2): Move parts without snap
- Rotate tool (3): Rotate parts
- Root tool (4): Change root part
- C: Toggle angle snap
- X: Toggle symmetry (cycle SPH/VAB mirror/radial modes)
- F: Cycle symmetry count (1x, 2x, 3x, 4x, 6x, 8x)
- Shift+click: Duplicate part subtree
- Alt+click: Copy part only (no children)
